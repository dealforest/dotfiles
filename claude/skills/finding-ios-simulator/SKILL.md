# iPhoneシミュレータの取得

最新iOSランタイムの無印iPhoneシミュレータを取得する。

## 実行

### iPhone
```bash
scripts/get-ipad-simulator.sh | cut -f1
```

### iPad
```bash
scripts/get-ipad-simulator.sh | cut -f1
```

## 出力例

```
iPhone (A16)
```
