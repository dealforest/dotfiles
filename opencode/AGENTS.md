# グローバル指示

このファイルはすべての OpenCode セッションで適用されるグローバルなルールです。

## 言語

- コミットメッセージは日本語で書く
- PR の説明文は日本語で作成する（特に指定がない限り）

## Git ワークフロー

- Git Flow ブランチ戦略に従う（`instructions/git-flow.md` を参照）
- 細かくコミットし、後から整理する（`instructions/git-commit.md` を参照）
- Conventional Commits 仕様に従う（`instructions/conventional-commits.md` を参照）

## スキル利用

タスクに応じて適切なスキルを積極的に活用すること。
