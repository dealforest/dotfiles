---
description: 最新iOSランタイムの無印iPhoneシミュレータ名を取得する
---

最新iOSランタイムの無印iPhoneシミュレータを取得する。

## 実行

### iPhone
```bash
~/.config/opencode/skills/finding-ios-simulator/scripts/get-iphone-simulator.sh | cut -f1
```

### iPad
```bash
~/.config/opencode/skills/finding-ios-simulator/scripts/get-ipad-simulator.sh | cut -f1
```

## 出力例

```
iPhone (A16)
```
